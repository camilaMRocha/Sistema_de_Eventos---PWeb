# Sistema de Eventos - _**HeyEvents**_ 

* Nome do projeto -> HeyEvents
* Tema escolhido -> Sistema de eventos

 O projeto tem como objetivo conectar clientes a locais, fornecedores e profissionais especializados, oferecendo uma plataforma centralizada para pesquisar serviços, simular orçamentos, realizar reservas e acompanhar todo o planejamento de eventos como casamentos, aniversários, formaturas e confraternizações.
